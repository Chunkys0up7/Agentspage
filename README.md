## How to Use

### For Presentations:

1. **Open `index.html` in any web browser**
2. **Full-screen mode** (F11) for presentation view
3. **Click through demos** during your talk
4. **Works offline** - no internet needed

### Presentation Flow:

```
1. Start with hero section - explain what AI agents are
2. Show the 3-step flow: Perceive â†’ Think â†’ Act
3. Go through "Big Picture" benefits
4. Present each agent type (7 cards)

7. Conclude with "Why This Matters"
```

### Interactive Elements:

**During Presentation:**
- Click agent cards to open interactive demos
- Type custom inputs to show real-time processing
- Use comparison scenarios to show side-by-side differences

- Users can explore each agent type at their own pace
- Try different inputs in demos
- Run learning agent multiple times to see improvement
- Switch between comparison scenarios

## Usage Tips

### Best Practices:
1. **Test all demos** before presenting
2. **Prepare example inputs** that work well
3. **Explain one agent type thoroughly** before moving on
4. **Use comparison section** to reinforce learning
5. **Leave time for audience questions** and interaction

### Suggested Examples to Try:
- **Reflex Agent:** "URGENT: Click here to verify your account!"
- **Model-Based:** Ask "What's my order status?" then "When will it arrive?"
- **Goal-Based:** "Book a flight from NYC to LA next Monday"
- **Utility-Based:** "Find the best route from home to airport"
- **Learning Agent:** Run same input 5+ times to see improvement
- **Knowledge Agent:** "What are the benefits of AI agents?"
- **Multi-Agent:** "Plan a marketing campaign for a new product"

### Keyboard Shortcuts:

- **F11** - Full screen (presentation mode)
- **Tab** - Navigate between interactive elements
