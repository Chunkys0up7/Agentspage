Framework Choice

React/Vue or Vanilla JS?

+
Chosen: Vanilla JavaScript
Reasoning:
  • Simpler to understand for non-technical audience
  • No build step or dependencies required
  • Better for educational content (see actual code)
  • Faster load times for presentation
Alternatives Considered: React, Vue.js

Text input or Pre-populated?

+

Concept Explanation

Technical or Analogies?

+
Chosen: Real-world Analogies
Reasoning:
  • Target audience: "non-technical, AI-curious"
  • Analogies build mental models quickly
  • Examples: "like a thermostat", "like a GPS"
  • More memorable than technical definitions
Alternatives Considered: Academic definitions, technical specs
